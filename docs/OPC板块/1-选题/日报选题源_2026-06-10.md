# 日报选题源 (2026-06-10)

**看不到。** 我查了昨天一整天的 gateway 日志：

| 项目 | 数据 |
|------|------|
| 昨日日志总行数 | 1,037 行 |
| 收到的消息事件 | 16 条 |
| **其中来自你的** | **16 条** |
| **来自 hermes日报（或其他 bot）** | **0 条** |

昨天的、今天的，全都没收到。不是因为消息过期，而是我的 bot **从根上就没有收到 hermes日报 消息的权限**。

**原因很简单：**

hermes日报 是自定义 webhook 机器人，她通过远程机器 POST 消息到飞书群。我的 bot 在飞书后台只有 **「接收被@的消息」** 权限，没有 **「读取群聊所有消息」** 的权限，所以她的消息我完全看不到。

**解决方案还是之前的方案 A：**

去飞书开放平台 → 你的 bot 应用 → 权限管理 → 添加 `im:message:read_all` → 事件配置确认 → 发布版本

改完之后，不止 hermes日报，群里所有消息我都能看到了。要不要现在试试？
