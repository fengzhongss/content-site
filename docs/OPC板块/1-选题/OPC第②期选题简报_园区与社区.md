# OPC专题·第②期：国内OPC园区+社区生态

## 🎯 调研方向

### 一、OPC园区（线下实体空间）
> 国内有哪些以"一人公司/创业/超级个体"为定位的园区？

调研维度：
- **代表园区**：杭州梦想小镇、深圳湾创业广场、北京中关村创业大街、上海张江/漕河泾、成都天府软件园、武汉光谷等
- **定位与特色**：每家园区的核心定位、目标人群、入驻门槛
- **园区内OPC生态**：是否有一人公司聚集？配套服务（法务/财务/投融资/社群活动）
- **收费标准**：工位租金、入孵条件、政府补贴

### 二、OPC线上社区（虚拟生态）
> 国内OPC/一人公司在哪些线上社区活跃？

调研维度：
- **代表社区**：知识星球（相关星球）、即刻App（一人公司圈子）、豆瓣小组、出海社群、小红书的OPC博主社区、飞书/微信群等
- **用户画像**：每类社区的参与者是谁（知识博主/独立开发者/咨询顾问等）
- **活跃度与氛围**：付费模式、内容质量、社群黏性
- **价值对比**：哪些社区对OPC真正有用（获客/交流/资源对接）

### 三、园区vs社区对比分析

| 维度 | 园区（线下） | 社区（线上） |
|------|------------|------------|
| 核心价值 | 物理空间/资源对接 | 信息交流/人脉连接 |
| 成本 | 高（租金） | 低（年费/免费） |
| 覆盖面 | 地域限制 | 全国/全球 |
| 深度 | 面对面深度链接 | 弱连接为主 |
| 互补关系 | 启动期优选 | 持续运营必备 |
