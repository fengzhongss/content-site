| 项目 | 内容 |

| 🔴 替代A（高铁） | **G字头高铁**（成都东→攀枝花南），约 **4.5小时**，二等座约 **¥280~¥320** |
| 🟡 替代B（大巴） | **成都石羊场客运站→西昌** 约6h，滚动发班，票价约 **¥150**；**西昌→攀枝花** 约3h，票价约 **¥70** |
| 🟢 替代C（分段/包车） | **成都→普雄**（K113次约7h，硬座¥70，一定要体验这最精华的一段）→**普雄住一晚**（便宜民宿约¥50-80）→**次日乘5633次到攀枝花**（8-9h慢车体验），这是最完整的慢车旅行方案 |
| ⚠️ 特殊提醒 | ① 🚨 **公益慢车条件艰苦**：无空调（夏季闷热、冬季有火炉但仍有凉意），厕所较简陋，部分车厢无广播 ② 停靠站点多，**注意防盗**，贵重物品随身 ③ **携带干粮和水**，部分区段7-8小时无补给站 ④ 5633次目前始发站为普雄，成都出发需先乘K113到普雄站 ⑤ 禁止对彝族同胞拍照时要有礼貌，先问再拍 |

### 🔹 出行安全须知

1. **购票**：公益慢车（5633次/5619次）部分站点不支持 12306 线上购票，建议提前到站购买或咨询当地车站
2. **车次动态**：西南地区（特别是凉山、贵州段）铁路线路受地质灾害影响较大，雨季出发前查询 @西南铁路 官方微博或12306公告
3. **高原安全**：拉林铁路虽全程供氧，但建议在拉萨/林芝适应1-2天后再乘其他线路；有心脏病/高血压/呼吸道疾病者咨询医生
4. **雨季（6-8月）**：四川凉山、贵州山区偶发泥石流/塌方造成线路中断，备选方案提前准备
5. **尊重习俗**：凉山彝族聚居区、藏南地区有独特民族习俗，拍照先征得同意，勿冒犯宗教场所
