### 1.1 西安→乌鲁木齐 兰新线

| 项目 | 内容 |
|------|------|
| 🚉 车次信息 | **K1352** / **Z40** / **T269** 等，西安站→乌鲁木齐站，途经宝鸡、天水、兰州、武威、张掖、酒泉、嘉峪关、哈密、吐鲁番 |
| ⏱ 运行信息 | 里程约 **2568km**，时长约 **25~32小时**（Z字头约25h，K字头约32h），硬座约 **¥302**，硬卧约 **¥531**，软卧约 **¥817** |
| 🌄 特色看点 | 兰新线是贯穿中国西北的大动脉，从关中平原一路向西穿越河西走廊。沿途可见黄土高原沟壑、祁连山雪山、戈壁荒漠、嘉峪关长城、哈密绿洲、吐鲁番火焰山等多样地貌。列车穿行在河西走廊"丝绸之路"核心段，沿途历史文化名城镇次第展开，是体验西北辽阔与丝路文明的最佳线路。 |
| 🔴 替代A（高铁） | **G字头高铁**（西安北→乌鲁木齐），约 **12小时**，二等座约 **¥950**，一等座约 **¥1500** |
| 🟡 替代B（大巴） | **西安→兰州** 约8h（班次密集），**兰州→乌鲁木齐** 约20h（每日班次有限，建议分段） |
| 🟢 替代C（分段/包车） | **西安→兰州**（高铁约3h ¥230）→**兰州→敦煌**（K9661夜车体验河西走廊）→**敦煌→乌鲁木齐**（包车或火车沿哈密方向） |
| ⚠️ 特殊提醒 | ① 春季（3-5月）河西走廊可能有沙尘暴，影响运行速度 ② 冬季吐鲁番-乌鲁木齐段偶有风雪晚点 ③ 长途绿皮建议备足食物和水，沿途站点停靠时间长可下车补充 |
| 📅 最佳季节 | **5-6月、9-10月**：春秋两季气候舒适，河西走廊油菜花（6月底）和金秋胡杨林（10月）都是绝佳风景 |

### 1.2 西宁→拉萨 青藏线

| 项目 | 内容 |
|------|------|
| 🚉 车次信息 | **Z6811** / **Z165** / **Z21**，西宁站→拉萨站，途经德令哈、格尔木、那曲，其中Z6811是青藏铁路"绿皮标杆" |
| ⏱ 运行信息 | 里程约 **1956km**，时长约 **21~22小时**，硬座约 **¥224**，硬卧约 **¥495**，软卧约 **¥781** |
| 🌄 特色看点 | 世界上海拔最高的铁路线，被誉为"天路"。翻越唐古拉山口（5072m），穿越可可西里无人区、羌塘草原，沿途可见藏羚羊、藏野驴等野生动物。青海湖、纳木错、昆仑山脉、措那湖等绝美自然风光令人震撼。列车在格拉段（格尔木-拉萨）全程供氧，是国内最受推崇的火车旅行路线。 |
| 🔴 替代A（高铁） | **无直达高铁**，西宁→格尔木有动车（约4.5h ¥230），格尔木→拉萨可乘供氧列车 |
| 🟡 替代B（大巴） | **西宁→拉萨大巴** 约24小时，每日1-2班，票价约¥500，但高原长途大巴舒适度差，不推荐 |
| 🟢 替代C（分段/包车） | **西宁→格尔木**（动车4.5h ¥230）→**格尔木→拉萨**（供氧列车约14h ¥395硬卧），在西宁提前适应高原，格尔木休整一天更安全 |
| ⚠️ 特殊提醒 | 🚨 **高反风险**：唐古拉山口海拔5072m，列车内全程**弥散式供氧**，每个铺位也有吸氧口。建议提前服用红景天（提前3-5天），西宁（2200m）停留1-2天适应。列车上有随车医生，出现严重高反可求助。**格尔木段（2800m→5200m）海拔拉升最快**，该段尽量勿入睡，保持清醒观察身体反应。 |
| 📅 最佳季节 | **6-9月**：夏季含氧量最高，高原天气相对稳定。7-8月青海湖边油菜花盛开，那曲草原一片碧绿。冬季冰封期部分路段雪大，不建议初次进藏者冬季走此线 |

| ⚠️ 特殊提醒 | 🚨 **极度不稳定**：哈罗铁路是**通勤车/路用列车**，主要服务铁路职工和钾盐矿工人。**🚫 12306无法购票**，仅能在哈密站窗口询问当日的通勤车次，且经常无固定时刻表。该线路**随时可能停运**或取消客运搭载。此外罗中地区气候极端，夏季地表温度60-70℃，冬季-30℃，周边无补给。**不建议普通游客冒然前往**，务必提前联系哈密站确认。 |
| 📅 最佳季节 | **9-10月**：避开极端高温和极寒，但即便如此仍要做好充分准备，带足水和食物 |
